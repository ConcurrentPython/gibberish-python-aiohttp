Gibberish Python
================

Setup Python Environment:
```
export PYENV_ROOT=pyenv
pyenv/bin/pyenv install
pyenv/bin/pyenv init
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Run the server:
```
python server.py
```

Check it out: [http://localhost:8080](http://localhost:8080)
